Improvisation X Composition

Metamor Teamwork/2024


This work is an A/V performance that visualizes acoustic data from live instrumental performance in real-time. Audio characteristics such as audio levels, frequency spectrum, and attack-decay received from the instruments via Max/MSP are analyzed and establish bidirectional communication with TouchDesigner using OSC protocol. The subtle dynamics of the performance are instantly mapped to visual parameters, generating visual expressions synchronized with the musical articulation. The bidirectional communication between Max/MSP and TouchDesigner enables visual changes to be fed back into the audio processing, constructing an organic performance space where the performer and visual system mutually influence each other. At the event venue, this created a highly unique live experience where the vitality of the performance merged with the improvisational nature of the visuals. Metamor チームワーク/2024


本作品は、ライブ演奏される楽器の音響データをリアルタイムで視覚化するA/Vパフォーマンスです。Max/MSPを介して楽器から受信したオーディオレベル、周波数スペクトラム、アタック・ディケイなどの音響特性を解析し、OSCプロトコルを用いてTouchDesignerと双方向通信を確立。演奏の微細なダイナミクスが即座にビジュアルパラメータへとマッピングされ、音楽の表情と同期した視覚表現が生成されます。 Max/MSPとTouchDesigner間の双方向通信により、視覚的な変化を音響処理にフィードバックすることも可能にし、演奏者と視覚システムが相互に影響し合う有機的なパフォーマンス空間を構築しました。イベント会場において、演奏の生命力と視覚の即興性が融合した、一回性の高いライブ体験を創出しました。

Resonance

AudioVisual Performance/2024


An A/V performance work presented at Senzoku Digital Art Project 2024. This work is an audiovisual piece linking Ableton Live and TouchDesigner via OSC protocol. Information such as audio levels and effect parameters is extracted in real-time from Ableton Live, a music production software, and transmitted to TouchDesigner. By directly mapping the received data to the generation and transformation of visual elements, an experience where sound and image are organically interconnected is created. Through bidirectional communication via OSC protocol, musical changes are instantly converted into visual expressions, visualizing even the subtle nuances of sound. This technical approach constructs an immersive audiovisual space where auditory and visual elements mutually influence each other.

"Humming" for Piano, Video and Electronics

A/V Performance / Live Electronics/2025


Performed as the opening act for Jon Balke.
This work is an audiovisual performance in which piano and live electronics engage in dialogue. A timeline system built in Max/MSP using event numbers establishes the foundational structure of the piece, while bidirectional communication via OSC protocol with TouchDesigner enables real-time visual responses to the performance. On the sonic level, the acoustic piano sound is processed within Max/MSP through FFT analysis, delay, reverb (freeze), and sound granulation, creating a multi-layered texture where acoustic resonance merges with electronic sound. The piano's timbre is expanded temporally and spatially, generating an immersive sonic environment. On the visual level, organic tree structures are generated using L-system algorithms, with instancing techniques in TouchDesigner creating complex and dynamic visuals. Visual imagery that grows and transforms in response to sonic data resonates with the improvisational nature of the piano performance, offering an experience where auditory and visual elements unite as one. A/V パフォーマンス / ライブ・エレクトロニクス/2025


本作品は、ピアノ演奏とライブ・エレクトロニクスが対話する視聴覚パフォーマンスです。Max/MSPで事前に構築したイベントナンバーシステムにより作品の基本的なタイムラインを設定し、OSCプロトコルを介したTouchDesignerとの双方向通信によって、演奏に呼応するリアルタイムな視覚表現を実現しました。 音響面では、ピアノの生音をMax/MSP内でFFT解析、ディレイ、リバーブ(フリーズ)、サウンド・グラニュレーションなどの処理を施し、アコースティックな響きと電子音響が融合する多層的なテクスチャーを構築。ピアノの音色が時間的・空間的に拡張され、没入感のある音響空間を創出しています。 視覚面では、L-systemアルゴリズムを用いて有機的な樹木構造を生成し、TouchDesigner上でインスタンシング技法を活用することで、複雑かつ動的なビジュアルを実現。

Fragments of memory

6ch. Tape Music/2022


A 6-channel tape music composition.
This work is a tape music piece created primarily using the Granulator in Ableton Live. Using environmental sounds collected through field recording and various sound samples as source material, the audio is reconstructed at a granular particle level by modulating parameters such as grain size and position. Leveraging the characteristics of granular synthesis, the work freely manipulates time stretching and timbral transformation while layering various effect processing, converting the original sound sources into entirely new acoustic spaces while retaining their inherent textures. Through a 6-channel configuration utilizing Max for Live, the sound image moves through space, realizing a three-dimensional soundscape that envelops the listener. 6ch テープ音楽


6チャンネルのテープ音楽作品/2022
本作品は、Ableton LiveのGranulatorを中心に制作したテープ音楽作品です。フィールドレコーディングで収集した環境音や様々なサウンドサンプルを素材とし、グレインサイズやポジションなどのパラメータを変化させることで、音響を微細な粒子レベルから再構築しました。 グラニュラー合成の特性を活かし、時間軸の伸縮や音色の変容を自在に操作しながら、各種エフェクト処理を重ねることで、元の音源が持つ質感を保ちつつも、全く新しい音響空間へと変換しました。Max for Liveを活用した6チャンネル構成により、音像が空間内を移動し、聴取者を包み込む立体的なサウンドスケープを実現しました。
